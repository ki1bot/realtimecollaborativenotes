# Realtime Collaborative Notes

Realtime Collaborative Notes adalah aplikasi catatan kolaboratif berbasis web yang memungkinkan pengguna membuat, mengelola, dan mengedit catatan secara real-time. Project ini menggunakan arsitektur full-stack dengan frontend React TypeScript dan backend Node.js Express yang terhubung ke MongoDB serta Socket.IO untuk komunikasi real-time.

## Fitur Utama

- Autentikasi pengguna dengan register dan login
- Manajemen catatan pengguna
- Editor catatan berbasis web
- Kolaborasi real-time menggunakan Socket.IO
- Proteksi route menggunakan autentikasi
- REST API untuk komunikasi frontend dan backend
- Penyimpanan data menggunakan MongoDB
- Struktur project terpisah antara frontend dan backend
